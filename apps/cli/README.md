# marchen-spec

Spec-driven development CLI.

## 安装

```bash
# 直接使用
npx marchen-spec init

# 或全局安装
npm i -g marchen-spec
marchen init
```

## 命令

```
marchen init [options]  初始化 MarchenSpec 目录结构
marchen new <name>      创建一个新的变更
marchen list            列出所有 open 状态的变更
marchen archive <name>  归档一个已完成的变更
```
